# MapStack

Simple, shareable map collections. Add places, customize markers, share with one link.

## Features

- ✅ Add places by address (auto-geocoded)
- ✅ Custom categories (ramen, onsen, hotel, etc.)
- ✅ Color-coded markers
- ✅ Rich notes per place
- ✅ Shareable links (data encoded in URL)
- ✅ Mobile-responsive
- ✅ Works offline (localStorage)
- ✅ No signup required

## Usage

1. Open `index.html` in browser
2. Click "Add Place"
3. Enter name, address, category, notes
4. Place appears on map
5. Click "Share" to get shareable link

## Tech Stack

- Pure HTML/CSS/JavaScript (no framework)
- Leaflet.js for maps
- OpenStreetMap tiles
- Nominatim for geocoding
- localStorage for persistence
- Base64 URL encoding for sharing

## Deploy

Can be hosted on:
- GitHub Pages (free)
- Vercel (free)
- Netlify (free)
- Any static host

Just upload the files!
